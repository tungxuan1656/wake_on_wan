Setup Wake on Lan

What is Wake on Lan and how to enable it

Setup router to Wake on Wan

  • Step 1: Port forwarding / Port mapping configuration
    • Add following forwarder: UDP, from 1007 to 9, ip 192.168.1.254
      • 192.168.1.254 this IP shouldn't belong to any device. It is a free IP address we will use for an ARP entry later
      • 1007 is the outoing port we opened on router.
      • 9 is the port my motherboard uses for wake on lan packages. Check manual for yours!
  • Step 2: Set ARP Table / ARP configuration
    • We will need to add ARP entry in the router:
      • Command: arp -i br0 -s 192.168.1.254 FF:FF:FF:FF:FF:FF
      • GUI: IP: 192.168.1.254 / MAC Address: FF:FF:FF:FF:FF:FF
